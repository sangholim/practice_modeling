## 자판기 입력값 정의서
### 운영 모드
- 입력값: 
``` jsonc
{
  "status": "RUNNING"
}
```


#### 구매
- 입력값: 
``` jsonc
{
   "status": "MANAGEMENT",
   "amount": 투입 금액,
   "drinks": 선택한 음료수 (list)
}
```

### 관리자 모드
- 입력값:
``` jsonc
{
  "status": "MANAGEMENT"
}
```
### 프로그램 종료
- 입력값:
``` jsonc
{
  "status": "SHUTDOWN"
}
```
