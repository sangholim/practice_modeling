### 분류 관리
- 분류 등록시 상위 분류 확인
- 이미 등록된 하위 카테고리 데이터를 다른 카테고리에서 참조해서 쓰면 안된다. 사용시 새로 카테고리 생성
```
a -> b -> c
a -> b -> c1

a -> b1 -> c2
a -> b1 -> c3

a -> b2 -> c4
a -> b2 -> c5

* a - b - c 등록된 카테고리가
* a - b1 -c `c` 하위 카테고리 참조를 통한 중복은 하지 않음
```
