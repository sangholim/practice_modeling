### 기업 가입 시나리오
- 관리자 화면에서 기업 가입 수행
- 가입시 integration 필요한 필드: 이메일 인증, 휴대폰 번호 인증, 증명서
- 증명서로는 하나의 기업만 가입 가능하다.
- 조직명: 10자 이하 한글, 영어
- 이메일: [영어,숫자]@[영어,숫자].[영어,숫자]
- 휴대폰 번호: 9~11 숫자 [- 없음]
- 인증서: NotBlank
- 주소: NotBlank
- 기업 가입 이후, 마스터 계정 추가

### 기업 관리자 계정 생성
- 회사 번호
- 사원명: 기업명
- 직책: 관리자
- 이메일: 기업 이메일
- 휴대폰 번호: 기업 휴대폰 번호
- 비밀번호: 임시 비밀번호 생성 (휴대폰 인증후 비밀번호 재설정)
- 권한: 기업 관리자

### 기업 조회 시나리오
- 필드: 이름, 이메일, 휴대폰 번호, 인증서 ,주소를 노출

### 기업 유효성 조건
- 증명서 중복 불가
- 조직명 중복 불가
